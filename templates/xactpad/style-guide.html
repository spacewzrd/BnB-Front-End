Helpers

Disable animation

                  
Add "ng-animate-disabled"  to any element to disable ALL animations on it.

Disable text decoration

                  
Add "txt-dec-none" to any element to disable text decoration.

Set element to Flex and Column

Ex:

thing1

thing2

                    
<div class="flexColumn">

                    
<p>thing1</p>

                    
<p>thing2</p>

                    
</div>

                    
Add "flexColumn" to any element to make a column with flexbox.

Set element to Flex and Row

Ex:

thing1

thing2

                    
<div class="flexRow">

                    
<p>thing1</p>

                    
<p>thing2</p>

                    
</div>

                    
Add "flexRow" to any element to make a row with flexbox.

Text Align

                    
<div class="center">

                    
<p>thing1</p>

                    
</div>

                    
Add "center", "text-align-left", or "text-align-right" to any element to set text-align

Pointer

                  
Add "pointer" to an element to set its mousetype to pointer

Margin Helpers

                    
Add "marg-(direction) to add 1em of margin to that side"

                    
Ex:

                    
marg-left

                    
marg-right

                    
marg-top

                    
marg-bot
